# whatweb-plus


已内测、已公测, 无缺陷性问题反馈, 代码开源, 放心使用.  ---NOVASEC 酒零

    whatweb自开始改造截至目前已更新10个版本以上，
    目前版本是0.5.5.12，是最新版，也是最终版，后续只会对插件进行增删和优化。
    这个版本后，我已将我目前对指纹识别工具所有的要求和理解都进行了实现。
    如果没有其他的同样功能支持的Go版本替代，这应该是最强大的指纹识别工具了。
